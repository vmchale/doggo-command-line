## Doggo command-line tool to evaluate woofers

### Installation

To install the easiest is probably

```
cargo install doggo
```

You can install cargo with

```
curl https://sh.rustup.rs -sSf | sh
```

### Usage

Find out if he/she is a good doggo.

```
$ doggo jethro -m

jethro is a good woofer.
```
